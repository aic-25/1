Bracelet tutorial

To make a bracelet like the one in the picture you will need the following:

    • Beads of your choice
      A clasp
      Many charms
      Jumprings
      Crimp & Needle Nose Pliers
      Cutters
      Eyepins 3cm(preferably)

    • First we will start by grabbing the eyepins and using the needle nose pliers to open them to insert the beads and charms, then using the crimp plier close them. Once this is done we will also do the same thing but use the jump rings instead and insert small charms that fit into them and with the pliers close them. After having everything done we just need to put it together, so once again using the needle nose and crimp pliers you will connect all the eyeheads with the beads inside them in a way that creates the design you want while also adding the jump rings with the beads to the eyheads. The picture above can be used as a reference for you to follow a pattern. Once the main part is connected, you simply need to close it which is when you will add the clasp at one of the ends of the bracelet by using the pliers and connecting it with a jump ring. On the other end of the bracelet you would need to add a jump ring(a bigger one if available) and then you will have created a bracelet all by yourself!

    Earring tutorial

    Making earrings is also a fun jewelry to make and is better for beginners as it's more simple and less time consuming! Here are the materials you will need:

    • 2 beads
      2 charms
      2 earring hooks
      Two earring backs
      Jump rings
      Needle Nose Pliers
      Cutters(optional)
      Eyepins(preferably) or headpins 2cm
      Cable chain (preferably thin)

    • To start off you will grab the bead/charm of your choice and insert it into an eyepin each using the pliers to open and close them. In the same way, you will then use the jump rings instead and use the pliers to open and close them aftet inserting a bead/charm into each of them. At this point you should have four pieces in total. Once this is done, you can then cut some cable chain with the cutters and add a jump ring on each side using the pliers to open and close them. Then we connect one side to the eyepin we previsoly worked on while on the other side you add the previously worked on jump ring with the bead/charm inside it. Finally, you should now have two pieces to which you will add an earring hook each using jump rings and pliers. Now you are all done and have made earrings!
